## Project Title
**TODJE – Online Watch Store**

---

## Project Description
TODJE is a modern e-commerce website developed using Shopify for selling stylish and premium wristwatches.  
The platform allows customers to browse watch collections, view product details, check prices, and simulate shopping cart interactions.

The website focuses on simplicity, elegance, and user-friendly navigation to improve customer shopping experience.

---

## Platform Used
- **Shopify**
- **GitHub** (for project documentation)

---

## Features Implemented
The website includes the following features:

- Homepage with welcome message
- Watch product collection display
- Product images, prices, and descriptions
- About Us page
- Shopping cart interaction
- Responsive mobile-friendly design
- Email subscription section
- Clean and modern UI/UX navigation
